WeSchedule!

About the Project

The WeSchedule was created to simplify the process of coordinating study sessions and meetings among groups of people. Inspired by the challenges of aligning schedules for student projects, this tool is designed to find overlapping time slots efficiently, accommodate event durations, and even generate Google Calendar events.

The ultimate goal of the project is to create an intuitive, user-friendly scheduling tool that integrates seamlessly with other productivity platforms like Google Calendar.

Key Features

  • Find overlapping availability for multiple participants.
  • Set a custom duration for events to get precise intervals.
  • Suggest alternative time slots for shorter overlaps.
  • Generate Google Calendar events and send invites via email.

Installation and Usage

Requirements

Ensure you have Python 3.8 or higher installed on your system.

Installation

    
# Clone the repository
git clone https://github.com/yourusername/study-group-scheduler.git
cd study-group-scheduler

# Install dependencies
pip install -r requirements.txt

Usage

Run the tool with the following command:

    python scheduler/scheduler.py

Example:

Enter availability for each participant, set an event title, choose a duration if needed, and get suggestions for overlapping times. If Google Calendar integration is enabled, you'll also receive an email invitation.
